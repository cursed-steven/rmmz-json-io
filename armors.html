    Armors.json

    タブ区切りテキストで書き出しますので、Excel等で読み込むときは区切り文字をタブに設定してください。カンマは入れちゃダメ。

    csvの各列の説明

    • ID: 防具IDです。
    • アイコン: ツクール本体側のアイコンの左下に小さく書いてある数字で指定します。
    • 説明: 改行を入れたい場合<改行>と記述します。
    • 防具タイプ・装備タイプ: それぞれタイプタブにあるIDで指定します。
    • 特徴: 編集しないでください。jsonに戻すとそのまま元データになります。
    • メモ: 改行を入れたい場合<改行>と記述します。
    • jsonファイルを指定して "SAVE AS CSV" を押すと、Armors.json を読み込んで Armors.csv を出力する。
      Save as csv
    • csvファイルを指定して "SAVE AS JSON" を押すと、Armors.csv を読み込んで Armors.json を出力する。
      Save as json
